K Klasio Retour à l'application

Vie privée

Klasio est conçu pour composer des classes sans compte, sans serveur applicatif et sans suivi utilisateur.

En bref : les données saisies dans Klasio restent dans votre navigateur, sauf si vous décidez de les exporter ou de partager un lien.

Données saisies

Klasio peut contenir le nom de l'école, l'année scolaire, les effectifs par niveau, les classes, les noms d'enseignant·es, les notes libres et les totaux filles/garçons si vous les renseignez.

Stockage local

Le travail est enregistré dans le localStorage du navigateur. Il reste donc sur l'ordinateur et le profil navigateur utilisés. Effacer les données du navigateur peut supprimer les brouillons locaux.

Fichiers .klasio

Le fichier .klasio est une sauvegarde JSON réimportable. Il contient la répartition au moment de l'export. Protégez-le comme tout document de travail de l'école.

Liens de partage

Le bouton Partager encode la répartition directement dans l'URL. Le lien n'est pas envoyé à Klasio, mais toute personne qui reçoit ce lien peut lire les informations qu'il contient.

Exports

Les exports PDF, TXT, CSV et mural sont générés dans le navigateur. Leur conservation, transmission ou impression dépend ensuite de votre usage local.

Réseau et services tiers

  • Aucun compte utilisateur n'est créé.
  • Aucun backend Klasio ne reçoit vos répartitions.
  • Aucun cookie de tracking ni outil d'analytics n'est intégré.
  • La police Inter peut être chargée depuis Google Fonts selon votre navigateur et votre cache.

Bonnes pratiques

  • Évitez de saisir des données nominatives d'élèves dans les notes libres.
  • Utilisez le fichier .klasio pour garder une sauvegarde maîtrisée.
  • Partagez les liens uniquement avec les personnes concernées.
  • Supprimez les brouillons du navigateur si vous travaillez sur un poste partagé.
