Importer une liste de niveaux

Collez une liste au format Niveau, Effectif — une ligne par niveau. Les niveaux déjà présents seront mis à jour, les nouveaux créés.

Brouillons A, B et C

Comment ça marche ?
  • Trois emplacements indépendants (A, B, C) : chacun garde ses propres effectifs, classes, notes et historique Ctrl+Z / Ctrl+Y. Tout est enregistré dans ce navigateur.
  • Actif : c’est le brouillon affiché dans l’assistant. Les autres restent en mémoire sans changer tant que vous ne les activez pas.
  • Copier vers… : duplique tout le contenu d’un slot vers un autre (utile pour partir d’une variante sans toucher à l’original).
  • Vider : remet un slot à zéro (effectifs à 0, plus aucune classe).
