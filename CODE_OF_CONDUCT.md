# Code de conduite

Klasio vise à rester un projet accueillant, utile et sobre pour les équipes éducatives et les personnes qui y contribuent.

## Comportements attendus

- Être respectueux, précis et constructif.
- Critiquer le code, les choix produit ou la documentation sans attaquer les personnes.
- Garder les échanges focalisés sur l'amélioration du projet.
- Signaler les désaccords avec des exemples concrets et des alternatives praticables.

## Comportements refusés

- Harcèlement, insultes, propos discriminatoires ou intimidants.
- Publication d'informations privées sans consentement.
- Spam, hors-sujet répété ou pression insistante sur les mainteneurs.
- Divulgation publique de vulnérabilités avant correction coordonnée.

## Signalement

Pour un problème de comportement, contactez le mainteneur via son profil GitHub. Pour une vulnérabilité de sécurité, suivez plutôt [SECURITY.md](./SECURITY.md).

Les mainteneurs peuvent masquer, fermer ou supprimer les contributions qui enfreignent ce code, et bloquer temporairement ou définitivement les personnes concernées si nécessaire.
