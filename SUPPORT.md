# Support

Klasio est un projet open-source maintenu publiquement sur GitHub.

## Besoin d'aide

- Pour utiliser l'application : commencez par la [version GitHub Pages](https://boblebol.github.io/Klasio/).
- Pour comprendre la confidentialité : consultez [`privacy.html`](./privacy.html).
- Pour un bug reproductible : ouvrez une issue avec le modèle **Signaler un bug**.
- Pour une idée produit : ouvrez une issue avec le modèle **Proposer une fonctionnalité**.

## Sécurité

Ne signalez pas une vulnérabilité dans une issue publique. Suivez la procédure de [`SECURITY.md`](./SECURITY.md).
